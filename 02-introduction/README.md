# Java简介

## Java分类
- JavaSE(Java 2 Platform Standard Edition) java标准版
- JavaEE(Java 2 Platform Enterprise Edition) java企业版
- JavaME(java 2 Platform Micro Edition) java微型版	

## 主要特性
- Java语言是简单的
- Java语言是面向对象的 类 接口 继承
- Java语言是分布式的
- Java语言是健壮的
- Java语言是安全的
- Java语言是体系结构中立的
- Java语言是可移植的
- Java语言是解释型的
- Java是高性能的
- Java语言是多线程的
- Java语言是动态的
